# Taller 3 - Gestión de Registros con Python y MongoDB

Este proyecto permite gestionar registros usando un menú en Python y guardar los datos en MongoDB.

## Características
- Agregar, consultar, modificar y eliminar registros.
- Conexión a MongoDB local (`mongodb://localhost:27017/`).
- Interfaz de menú interactivo.

## Requisitos
- Python 3.x
- MongoDB
- Librería `pymongo` (`pip install pymongo`)

## Cómo ejecutar
1. Asegúrate de que MongoDB esté corriendo.
2. Ejecuta el programa:
```bash
python main.py
